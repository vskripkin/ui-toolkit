Buttons

С легкостью создавайте кнопки!


Наши кнопки принимают 8 зарезервированых параметров:


  • @bg-color - цвет фона
  • @color - цвет текста
  • @hover - цвет фона при наведении
  • @font-size - размер шрифта
  • @padding-ver - внутренний отступ по вериткали
  • @paddong-hor - внутренний отступ по горизонтали
  • @border-width - ширина рамки
  • @pre - class для преффикса, стандартное значение "__"
.my-btn { #ui.button( @bg-color, @color, @hover,  @font-size, @padding-ver, @paddong-hor, @border-width, @pre ); }
