Inputs

С легкостью создавайте поля ввода


Наши input'ы принимают 6 зарезервированых параметров:


  • @font-size - размер шрифта
  • @margin-bottom - внешний отступ снизу
  • @padding-top - внутрений отступ сверху
  • @padding-bottom - внутрений отступ снизу
  • @padding-hor - внутрения отступ по горизонтали(слева и справа)
  • @prefix - class для преффикса, стандартное значение "__"
.my-input { #ui.textfield ( @font-size, @margin-bottom, @padding-top, @padding-bottom, @padding-hor, @prefix ); }

Input

html code

<input type="text" class="textfield" placeholder="demo text" />

less code

.textfield
{
	#ui.textfield(@margin-bottom: 1rem);

	.states() // описываем состояния
	{
		.placeholder() // стили для placeholder
		{
			color: #777;
		}
		.default() // обычное состояние
		{
			color: #333;
			border-width: 0 0 1px;
			border-style: solid;
			border-color: rgba(0,0,0,0.15);
			box-shadow: none;
		}
		.focus() // состояние при фокусе
		{
			border-color: @color-primary;
			box-shadow: inset 0 -1px 0 0 @color-primary;
		}
		.valid() // состояние :valid
		{
			border-color: @color-success;
			box-shadow: inset 0 -1px 0 0 @color-success;
		}
		.invalid() // состояние :invalid
		{
			border-color: @color-error;
			box-shadow: inset 0 -1px 0 0 @color-error;
		}
		.disabled() // состояние disabled
		{
			color: #AAA;
			border-color: rgba(0,0,0,0.45);
			box-shadow: none;
		}
		.readonly() // состояние readonly
		{
			
		}
	}
}
