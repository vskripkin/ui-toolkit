Buttonset

С легкостью создавайте сеты кнопок!


Наши сеты принимают 3 зарезервированых параметра:


  • @border-width - ширина рамки у дочерних элементов. на это значение делается отрицательный margin-top или margin-left, в зависимости от направления
  • @dir - направление. примает значение hor или ver
  • @prefix - class для преффикса, стандартное значение "__"
.my-buttonset { #ui.buttonset ( @border-width, @dir, @prefix ); }

Горизонтальное положение

html code

<div class="btns-line-1 btnset-line-1 __hor">
	<button data-wave="true" class="btn1">Button 1</button>
	<button data-wave="true" class="btn2">Button 2</button>
	<button data-wave="true" class="btn3">Button 3</button>
	<button data-wave="true" class="btn4">Button 4</button>
	<button data-wave="true" class="btn5">Button 5</button>
</div>

less code

.btnset-line-1.__hor { #ui.buttonset(@dir: hor); }

result
