Rangefield

С легкостью создавайте range!


Наши range принимают 6 зарезервированых параметра:


  • @font-size - размер шрифта
  • @track-height - высота ...
  • @thumb-width - ширина ...
  • @thumb-height - высота ...
  • @thumb-offset - высота ...
  • @prefix - class для преффикса, стандартное значение "__"
.my-rangefield { #ui.rangefield ( @font-size, @track-height, @thumb-width, @thumb-height, @thumb-offset, @prefix ); }

Rangefield

html code

<input type="range" class="rangefield" />

less code

.rangefield
{
	#ui.rangefield(
		@track-height: 1rem,
		@thumb-width: 2.5rem,
		@thumb-height: 2.5rem,
		@thumb-offset: 1px
	);

	.track() // описываем состояния для track
	{
		.default() // обычное состояние
		{
			background-color: @color-primary;
			border-radius: 5px;
			transition: background-color 300ms;
		}
		.hover() // состояние при наведении курсора
		{
			background-color: tint(@color-primary, 25%);
		}
		.focus() // состояние при фокусе
		{
			background-color: tint(@color-primary, 25%);
		}
		.disabled() // состояние disabled
		{

		}
	}

	.thumb() // описываем состояния для thumb
	{
		.default() // обычное состояние
		{
			color: @color-primary;
			border-radius: 50%;
			transition: color 300ms;
		}
		.hover() // состояние при наведении курсора
		{
			color: darken(@color-primary, 5%);
		}
		.focus() // состояние при фокусе
		{
			color: darken(@color-primary, 5%);
		}
	}
}
