Buttons

С легкостью создавайте кнопки!


Наши кнопки принимают 8 зарезервированых параметров:


  • @bg-color - цвет фона
  • @color - цвет текста
  • @hover - цвет фона при наведении
  • @font-size - размер шрифта
  • @padding-ver - внутренний отступ по вериткали
  • @paddong-hor - внутренний отступ по горизонтали
  • @border-width - ширина рамки
  • @pre - class для преффикса, стандартное значение "__"
.my-btn { #ui.button( @bg-color, @color, @hover,  @font-size, @padding-ver, @paddong-hor, @border-width, @pre ); }

Buttonset

С легкостью создавайте сеты кнопок!


Наши сеты принимают 3 зарезервированых параметра:


  • @border-width - ширина рамки у дочерних элементов. на это значение делается отрицательный margin-top или margin-left, в зависимости от направления
  • @dir - направление. примает значение hor или ver
  • @prefix - class для преффикса, стандартное значение "__"
.my-buttonset { #ui.buttonset ( @border-width, @dir, @prefix ); }

Checkfield

С легкостью создавайте radio и checkbox


Наши radio/checkbox принимают 5 зарезервированых параметров:


  • @type - тип элемента, принимает значение radio или checkbox
  • @font-size - размер шрифта
  • @pseudo - class для псеудо, стандартное значение "pseudo"
  • @label - class для лейбла, стандартное значение "label"
  • @prefix - class для преффикса, стандартное значение "__"
.my-checkfield { #ui.checkfield ( @type, @font-size, @pseudo, @label, @prefix ); }

		}

		}

.radio
{
	#ui.checkfield(radio);

	.states() // описываем состояния
	{
		.default() // обычное состояние
		{
			border-radius: 50%;
			box-shadow: inset 0 0 0 0.1em currentColor;
			transition: color 0.3s cubic-bezier(0.23,1,0.32,1);
			will-change: color;
			color: @color-primary;

			&:after
			{
				width: 100%;
				height: 100%;
				transition: transform 0.3s cubic-bezier(0.23,1,0.32,1),
							opacity   0.3s cubic-bezier(0.23,1,0.32,1);
				will-change: transform, opacity;
				transform: scale(0);
				opacity: 1;
			}
		}
		.hover() // состояние при наведении курсора
		{
			&:after
			{
				opacity: 0.5;
				transform: scale(0.45);
			}
		}
		.checked() // состояние checked
		{
			&:after
			{
				opacity: 1 !important;
				transform: scale(0.5) !important;
			}
		}
		.disabled() // состояние disabled
		{
			opacity: 0.4;

			&:after
			{
				opacity: 0;
			}
		}
		.disabled-checked() // состояние disabled checked
		{
			opacity: 0.4;
		}
	}
}

		.disabled() // состояние disabled

		}
	}

	{

Inputs

С легкостью создавайте поля ввода


Наши input'ы принимают 6 зарезервированых параметров:


  • @font-size - размер шрифта
  • @margin-bottom - внешний отступ снизу
  • @padding-top - внутрений отступ сверху
  • @padding-bottom - внутрений отступ снизу
  • @padding-hor - внутрения отступ по горизонтали(слева и справа)
  • @prefix - class для преффикса, стандартное значение "__"
.my-input { #ui.textfield ( @font-size, @margin-bottom, @padding-top, @padding-bottom, @padding-hor, @prefix ); }

Selectfield

С легкостью создавайте select!


Наши select'ы принимают 7 зарезервированых параметров:


  • @font-size - размер шрифта
  • @margin-bottom - внешний отступ снизу
  • @padding-top - внутрений отступ сверху
  • @padding-bottom - внутрений отступ снизу
  • @padding-hor - внутрения отступ по горизонтали(слева и справа)
  • @prefix - class для преффикса, стандартное значение "__"
  • @arrow - стрелка, стандартное значение "default"
.my-selectfield { #ui.selectfield ( @font-size, @margin-bottom, @padding-top, @padding-bottom, @padding-hor, @prefix, @arrow ); }
